Reflection

Building this Doritos tier list helped me understand how to properly structure and style tables in HTML and CSS. I learned how to use thead, tbody, and scope attributes to make the table more accessible. I also practiced adding and aligning images in table cells and applying color-coded tiers using CSS classes. The most challenging part was balancing readability and visual design. I validated both HTML and CSS using W3C tools and reviewed examples from W3Schools to guide my structure.
